# Case studies

Three long-form worked examples that put every previous chapter to use:

- **[Epilepsy surgery](epilepsy.md)** — multi-modal fusion (MRI + EEG + PET + neuropsych) for the surgical-candidacy decision in drug-resistant focal epilepsy. The canonical PrecisionStack example.
- **[Oncology — immunotherapy response](oncology.md)** — molecular + clinical stratification for checkpoint-inhibitor benefit in melanoma.
- **[Alzheimer's progression subtypes](alzheimers.md)** — SuStaIn-style disease-progression subtyping on ADNI, with implications for trial enrichment.

Each case study is intended to be readable in one sitting and ends with a reproducible task list a research engineer can pick up.
