# Foundations

Three chapters that ground the rest of the handbook:

- **[Patient heterogeneity](patient-heterogeneity.md)** — the twelve axes along which patients differ, and which precision-medicine task each axis matters for.
- **[Data modalities](data-modalities.md)** — what EHR, omics, imaging, wearables, and patient-reported outcomes look like as data, how they get joined, and the modality-specific failure modes.
- **[Bias and confounding](bias-and-confounding.md)** — the systematic distortions in clinical data, how each one corrupts which task, and the named mitigation strategies for each.

If you have a strong statistics background, you can skim *Patient heterogeneity* and *Data modalities*, but read *Bias and confounding* in full — it is the chapter that makes the difference between a model that gets published and a model that gets deployed.
