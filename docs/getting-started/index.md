# Getting started

This is the plain-language layer. Three short chapters, each readable in 10–15 minutes:

1. **[What precision medicine actually is](what-is-precision-medicine.md)** — the right treatment, the right patient, the right time. Why "average" is not enough, and where AI fits.
2. **[The patient-as-a-trajectory model](patient-trajectory.md)** — the single mental model that everything else in the handbook depends on. Patients are not snapshots; they are trajectories through a high-dimensional state space.
3. **[What AI can and cannot answer](what-ai-can-answer.md)** — the difference between prediction, explanation, and decision. A clinically useful AI model has to be honest about which of these it is doing.

If you've read these three chapters and feel comfortable with the vocabulary, you're ready for [Foundations](../foundations/index.md). If you'd rather see a worked example first, jump to [Epilepsy surgery](../case-studies/epilepsy.md) and come back.
